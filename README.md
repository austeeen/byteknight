# smoke

Project Smoke, or the "SmokeEngine", is a new project that hopes to produce a series of simple and
useful modules for building 2D games. It is designed to work alongside SFML 2.5.

## modules

 - events
    - node
    - network

- game
   - engine
   - game manager
   - scene manager
   - contexts
   - window

 - gui
    - widgets
    - guis
    - elements

 - io
    - devices
    - controller

 - sys
    - benchmark
    - exceptions
    - logging
    - core

 - utils
    - rect
